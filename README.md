# Cómo-funcionan-las-b-vedas-de-datos-patrimoniales-el-resguardo-de-riquezas-en-la-era-digital
Las  bóvedas de datos patrimoniales  representan un modelo de seguridad que ganó gran terreno entre los clientes de altos patrimonios. 
Las  bóvedas de datos patrimoniales se convirtieron en un auge respecto al resguardo ante la digitalización de activos y la ciberseguridad, al plantear una solución para la preservación y seguridad del patrimonio de individuos y familias de alto valor neto
Este modelo plantea la combinación de tecnología avanzada con  protocolos de seguridad,  y redefine  la forma en que se resguardan documentos, activos digitales y estrategias de planificación financiera.
